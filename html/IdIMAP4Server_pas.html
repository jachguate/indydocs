Contains constants, types, classes, and methods needed to implement an IMAP4 (Internet Message Access Protocol version 4.1) server.

Description

IdIMAP4Server.pas contains constants, types, classes, and methods needed to implement an IMAP4 (Internet Message Access Protocol version 4.1) server, as described in the Internet standards document: 

RFC 3501 - Internet Message Access Protocol version 4.1
