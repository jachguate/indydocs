Timeout for the protocol.

Pascal
property Timeout: Integer;
Description

Timeout is an Integer property that represents the number of milliseconds to wait for a protocol response from the time server. 

The default value for Timeout is TIME_TIMEOUT, as assigned during initialization of the component. 

Set the values for Host, Port, and Timeout prior to accessing DateTime, DateTimeCard, or SyncTime.
