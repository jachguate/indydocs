IdReplyPOP3.pas
Collapse All

IdReplyPOP3.pas contains Classes, Types, and Constants required to implement the POP3 Service Extension for Returning Enhanced Status Codes, 

as described in the Internet Standards documents: 

RFC 2449 - POP3 Extension Mechanism  

RFC 3206 - The SYS and AUTH POP Response Codes
