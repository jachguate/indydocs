IdDNSResolver.pas
Collapse All

Contains Classes, Types, Procedures, and Functions needed to define and implement a resolver for DNS protocol queries.

Description

IdDNSResolver.pas contains Classes, Types, Procedures, and Functions needed to define and implement a resolver for DNS protocol queries. The DNS protocol is described in the Internet Standards documents: 

RFC 1034 - Domain Names Concepts and Facilities  

RFC 1035 - Domain Names Implementation and Specification  

RFC 1591 - Domain Name System Structure and Delegation  

RFC 1183 - New DNS RR Definitions  

RFC 1884 - IP Version 6 Addressing Architecture  

RFC 1886 - DNS Extensions to support IP version 6  

RFC 1995 - Incremental Zone Transfer in DNS  

RFC 2181 - Clarifications to the DNS Specification.
