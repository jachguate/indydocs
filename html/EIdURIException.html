EIdURIException Class
Collapse All

Exception raised when an error in encountered in URI component values.

File

IdURI

Hierarchy
Pascal
EIdURIException = class(EIdException);
Description

EIdURIException is a EIdException descendant that represents an exception raised when an error in encountered while building the contents of the URI from it's components in TIdURI.GetFullURI.
