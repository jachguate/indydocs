IdIPMCastServer.pas
Collapse All

Contains classes needed to implement an IP Multicast server.

Description

IdIPMCastServer.pas contains classes needed to implement an IP Multicast server, as described in the Internet Standards documents: 

RFC 1112 - Host Extensions for IP Multicasting  

RFC 1458 - Requirements for Multicast Protocols
