IdFSP.pas
Collapse All

Contains classes, types, and constants needed to implement the File Service Protocol (FSP) version 2.

Description

IdFSP.pas contains classes, types, and constants needed to implement the File Service Protocol (FSP) version 2, as described in the specification document: 

File Service Protocol (FSP) version 2
