TIdTask.DoAfterRun Method
TIdTask Class   TIdTask Members
Collapse All

Triggers execution of the event handler for the task.

Pascal
procedure DoAfterRun;
Description

DoAfterRun is a procedure that performs the protected method that triggers execution of the event handler for the task. 

DoAfterRun, DoBeforeRun, and DoRun protect against the need to call inherited methods impelmented in descendant classes.
