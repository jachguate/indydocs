TIdDICT.Password Property
TIdDICT Class   TIdDICT Members
Collapse All

Authentication credentials provided during authentication for the client.

Pascal
property Password;
Description

Password is a published property in TIdDICT that represents the authentication credentials provided during authentication for the client. 

Username and Password are provided to the DICT AUTH command or the SASLMechanisms for the client depending on the values in AuthType and Capabilities.
