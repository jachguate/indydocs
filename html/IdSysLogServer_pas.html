IdSysLogServer.pas
Collapse All

Contains types and classes needed to implement a UDP-based BSD Syslog protocol server.

Description

IdSysLogServer.pas contains types and classes needed to implement a UDP-based BSD Syslog protocol server, as specified in the Internet Standards document: 

RFC 3164 - The BSD syslog Protocol
