ID_ACCEPT_WAIT Constant
Collapse All

Default timeout value used when accepting connections in TIdSimpleServer.

File

IdSimpleServer

Pascal
ID_ACCEPT_WAIT = 1000;
Description

ID_ACCEPT_WAIT is an Integer constant that represents the default timeout value used when accepting new connections in TIdSimpleServer. 

ID_ACCEPT_WAIT is used as the default for the TIdSimpleServer.AcceptWait property.
