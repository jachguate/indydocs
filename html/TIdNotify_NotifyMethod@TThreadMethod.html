TIdNotify.NotifyMethod@TThreadMethod
Collapse All

Performs a notification message using a thread method.

Parameters
Parameters 
Description 
AMethod 
Thread method to perform for the notification message. 
Description

NotifyMethod is a class procedure used to perform notification messages using the thread method specified in AMethod. 

NotifyMethod creates an internal TIdNotifyMethod instance and executes its Notify method. 

Use Notify to perform notifications using methods in the main thread for the application.
