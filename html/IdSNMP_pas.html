IdSNMP.pas
Collapse All

Contains types and classes needed to implement a Simple Network Management Protocol (SNMP) client.

Description

IdSNMP.pas contains types and classes needed to implement a Simple Network Management Protocol (SNMP) client, as described in the Internet Standards document: 

RFC 1157 - A Simple Network Management Protocol [SNMP]  

The Internet Activities Board recommends that all IP and TCP implementations be network manageable. This implies implementation of the Internet MIB, as described in the Internet Standards documents: 

RFC 1155 - Structure and Identification of Management Information for TCP/IP-based Internets  

RFC 1156 - Management Information Base for Network Management of TCP/IP-based internets
