Description

TIdMessageEncoderInfoQuotedPrintable is a TIdMessageEncoderInfo descendant that implements a class for representing information about registered encoder classes using Quoted Printable transfer encoding syntax. 

TIdMessageEncoderInfoQuotedPrintable reimplements the constructor for the class to set the class reference required to create instances of the encoder.
