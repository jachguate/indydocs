IdFTPServer.pas
Collapse All

Contains Classes, Types, and Constants needed to implement a multithreaded FTP server framework.

Description

IdFTPServer.pas contains Classes, Types, and Constants needed to implement a multithreaded FTP server framework, as defined in the Internet Standards documents: 

RFC 959 - File Transfer Protocol [FTP]  

RFC 2228 - FTP Security Extensions  

RFC 2389 - Feature negotiation mechanism for the File Transfer Protocol  

RFC 2428 - FTP Extensions for IPv6 and NATs  

RFC 2640 - FTP Internalization  

IETF Draft Extensions to FTP  

Deflate transmission mode for FTP  

Securing FTP with TLS
