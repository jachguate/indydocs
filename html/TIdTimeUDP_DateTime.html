TIdTimeUDP.DateTime
Collapse All

Current date and time from the remote time server.

Description

DateTime is a read-only TDateTime property that represents the current time according to the time server specified in the Host and Port properties. 

Reading the value in DateTime causes the client to to access the DateTimeCard value returned by the remote server. The cardinal value is converted to the native date and time value for the property, and adjusted to reflect the timezone bias for the local system. 

DateTime is used in SyncTime.
