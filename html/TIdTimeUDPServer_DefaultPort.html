TIdTimeUDPServer.DefaultPort Property
TIdTimeUDPServer Class   TIdTimeUDPServer Members
Collapse All

Port number for the Time protocol.

Pascal
property DefaultPort: integer;
Description

DefaultPort is an published property that represents the port number used to listen for client datagrams, as defined in the Time protocol. 

DefaultPort is redefined in TIdTimeUDPServer to establish the default port number for the protocol (IdPORT_TIME). 

DefaultPort is used to assign the default value used when creating listener threads defined in Bindings.
