TIdStreamVCL
Collapse All

TIdStreamVCL implements the interface used for the stream implementation on the platform or Operating System hosting the Indy library. 

 

For the .Net platform, TIdStreamVCL is a TIdStreamVCLDotNet descendant. 

 

For all other platforms (including Windows and Linux), TIdStreamVCL is a TIdStreamVCLWin32 descendant.
