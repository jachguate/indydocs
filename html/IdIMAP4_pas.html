IdIMAP4.pas
Collapse All

Contains constants, types, classes, and methods needed to implement an IMAP4 (Internet Message Access Protocol version 4.1) client.

Description

IdIMAP4 contains constants, types, classes, and methods needed to implement an IMAP4 (Internet Message Access Protocol version 4.1) client, as described in the Internet standards document: 

RFC 3501 - Internet Message Access Protocol version 4.1
