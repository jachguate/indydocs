IdReplySMTP.pas
Collapse All

IdReplySMTP.pas contains Classes, Types, and Constants required to implement the SMTP Service Extension for Returning Enhanced Error Codes, as described in the Internet Standards documents: 

RFC 1869 - SMTP Service Extensions  

RFC 2034 - SMTP Service Extension for Returning Enhanced Error Codes
