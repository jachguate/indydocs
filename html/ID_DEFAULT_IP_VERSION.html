ID_DEFAULT_IP_VERSION Constant
Collapse All

Default value for the IP version number supported in protocol stack methods.

File

IdGlobal

Pascal
ID_DEFAULT_IP_VERSION = Id_IPv4;
Description

ID_DEFAULT_IP_VERSION is a constant that represents the IP version number to use as the default value for arguments in stack-related method calls. ID_DEFAULT_IP_VERSION indicates the format to expect for IP address used in the protocol stack methods.
