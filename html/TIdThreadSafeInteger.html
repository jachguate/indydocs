Implements a thread-safe Integer data type.

File

IdThreadSafe

Hierarchy
Pascal
TIdThreadSafeInteger = class(TIdThreadSafe);
Description

TIdThreadSafeInteger is a TIdThreadSafe descendant that implements a thread-safe data type for storing an Integer value. 

TIdThreadSafeInteger relies on TIdCriticalSection for resource protection in multi-threaded applications. 

TIdThreadSafeInteger implements the Value property for reading or writing the Integer value for the data type using resource protection. 

TIdThreadSafeInteger also implements methods that protect write access to Value using Increment and Decrement.
